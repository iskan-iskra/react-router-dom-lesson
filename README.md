# React-Router-DOM v6 lesson

## Версии окружения (Versions):

- node: 20.11.1
- npm: 10.2.4

## Команды (Comands):

### установить зависимости (install dependencies)

```
npm install
```

### запустить локально проект (run project localy)

```
npm run dev
```

### собрать билд проекта (build dist for deploy)

```
npm run build
```
